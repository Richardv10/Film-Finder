A simple web application that suggests random movies based on your selected genre.

## Features

- Browse movies by genre
- Get random movie suggestions with poster, title, and overview
- Like or dislike movies to get new suggestions
- Clean, responsive interface

## How It Works

1. Select a movie genre from the dropdown menu
2. Click "Let's Play!" to get a random movie suggestion
3. Use the like/dislike buttons to get more suggestions

## API

This app uses [The Movie Database (TMDb) API](https://www.themoviedb.org/documentation/api) to fetch:
- Available movie genres
- Movies filtered by genre
- Detailed movie information including posters and descriptions

## Technologies

- HTML/CSS/JavaScript
- TMDb API
- Font Awesome icons
